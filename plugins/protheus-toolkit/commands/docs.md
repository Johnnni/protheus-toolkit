Invoque o agent `docs-reference` passando o termo de busca do usuario. O agent buscara em 3 camadas: inline → referencia expandida → MCP TDN.

Se o usuario nao especificou um termo, pergunte o que procurar (funcao, tabela, campo, PE, etc).
