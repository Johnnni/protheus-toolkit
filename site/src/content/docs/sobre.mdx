O **protheus-toolkit** é um plugin para [Claude Code](https://docs.anthropic.com/en/docs/claude-code) que transforma o assistente de IA num especialista em desenvolvimento TOTVS Protheus ADVPL/TLPP. Ele injeta conhecimento especializado, padrões de código e ferramentas de automação diretamente no contexto do Claude, permitindo que você desenvolva, analise e documente sistemas Protheus com muito mais produtividade.

## Como as skills funcionam

As skills são ativadas **automaticamente** com base no contexto da conversa. Quando você menciona um tema relacionado a Protheus — como MVC, ExecAuto, pontos de entrada ou consultas SQL — o Claude Code carrega a skill correspondente sem necessidade de comandos manuais.

Por exemplo, ao perguntar "como criar uma rotina MVC master-detail", a skill de MVC é ativada e injeta no contexto:

- Estrutura completa do modelo, view e controller
- Padrões de código com boas práticas
- Exemplos reais de implementação
- Armadilhas comuns e como evitá-las

## Validação automática com agents

Os agents do protheus-toolkit incluem um **checklist de auto-validação com 13 pontos** que é executado automaticamente ao final de cada geração de código. Esse checklist verifica:

- Tratamento de erros e exceções
- Controle de transações (`BEGIN TRANSACTION` / `END TRANSACTION`)
- Fechamento de queries e objetos
- Uso correto de variáveis `Static`, `Local` e `Private`
- Conformidade com padrões de nomenclatura TOTVS
- Proteção contra SQL injection
- E outros critérios essenciais para código de produção

Essa validação garante que o código gerado siga as melhores práticas do ecossistema Protheus desde o primeiro rascunho.
