Invoque o agent `code-reviewer` passando o contexto fornecido pelo usuario (arquivo, selecao, ou codigo colado). O agent detectara o tipo de codigo e aplicara as regras relevantes automaticamente.

Se o usuario nao especificou um arquivo, pergunte qual arquivo ou codigo revisar.
